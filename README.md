# Storyboard Skill

A Claude Code skill to generate infographic storyboard posters and video prompts for short videos (under 60 seconds).

## What it does

Given a video concept, this skill produces:

1. **Storyboard Prompt** — a complete prompt for image-gen AI to create a 16:9 infographic storyboard poster with scene-by-scene panels, cinematography notes, and a header/footer.
2. **Video Prompt** — a concise prompt for AI video apps (Seedance, Kling, Runway, Pika) with per-scene camera and action descriptions.

## Supported video types

- Food / Cooking
- Character Showcase
- Product Showcase
- Process / How-It-Works

## Language versions

- [English](storyboard-skill-en.md)
- [Tiếng Việt](storyboard-skill-vn.md)

## Usage

Install the skill and trigger it by mentioning: storyboard, shot breakdown, scene planning, video prompt, short video, reel planning, shooting script, shot list, or asking to visualize a video concept scene by scene.
